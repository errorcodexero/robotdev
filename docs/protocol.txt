DriverStation->Robot
--------------------------
0: PacketIndexHigh
1: PacketIndexLow
2: Header (cTagGeneral)
3: control code
4: request code
5: station code
N: time zone stuff
M: joystick stuff

Robot->DriverStation
----------------------------
0: PacketIndexHigh
1: PacketIndexLow
2: Header
3: control code   (cEmergencyStop)
4: rstatus code   (cHasRobotCode)
5: upper voltage
6: lower voltager
7: request code


Early: 
------
00 01 01 00 00 00
